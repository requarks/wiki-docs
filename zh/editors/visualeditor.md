# 概述

可视化编辑器非常适合非技术用户，他们更喜欢直接直观地表示他们正在编写的内容。它基于流行的[CKEditor 5](https://ckeditor.com/)软件。

# 用户指南

All formatting is done via the toolbar located above the editing area.

# 概述

> AsciiDoc编辑器在**2.5.295及以上版本**可用。
{.is-info}

AsciiDoc是一种轻量级的语义标记语言，主要用于编写技术文档。该语言可用于生成各种表示丰富的输出格式，所有这些格式都来自以简洁、可读、纯文本格式编码的内容。

# 使用

更多信息参见[AsciiDoc语言文档](https://docs.asciidoctor.org/asciidoc/latest/)。

> 请注意，目前只支持有限的AsciiDoc功能集。
{.is-warning}
